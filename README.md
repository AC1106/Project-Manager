##專案名稱：Project Manager
一個簡單的專案管理程式，用於管理團隊成員、專案和專案進度。


##概述
這個專案是一個基於命令行的專案管理應用程式，旨在幫助團隊組織管理專案信息和進度。使用者可以添加新專案、更新專案進度、查詢特定人員負責的所有專案清單以及刪除專案等功能。


##功能
添加新專案：使用者可以輸入專案的名稱、描述、進度和負責人，並將其添加到專案列表中。
更新專案進度：使用者可以更新現有專案的進度，並記錄最後編輯者。
查詢特定人員負責的所有專案清單：使用者可以查詢特定成員負責的所有專案，並查看其詳細信息。
刪除專案：使用者可以刪除現有的專案。


##如何使用
使用 C++ 編譯器編譯 Project Manager.cpp 文件。
運行編譯後的可執行文件。
根據命令行提示進行操作。
請選擇要進行的操作：
1. 建立新專案進度
2. 更新專案進度
3. 刪除專案
4. 查詢特定人員負責的所有專案List資訊
5. 離開
